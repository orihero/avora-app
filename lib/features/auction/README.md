# Auction Feature

Auction tab and flows: voting, participation, live bidding, winner/fallback (SOW §1–17).

## Push notifications (later phase)

Per SOW §14, push notifications are mandatory for:

- Auction start
- Approval / decline
- Auction pause / resume
- Winner and fallback status

SMS and email are out of scope for MVP. Implement push in a separate task after core flows are stable.
